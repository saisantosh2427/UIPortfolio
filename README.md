# Sai Santhosh UX Portfolio

A simple React + Figma-style UX portfolio based on the resume.

## How to run

```bash
npm install
npm run dev
```

## Sections included

- Hero section
- UX skills
- Interactive experience tabs
- Portfolio case study cards
- Design approach cards
- Contact section

## Suggested Figma portfolio pages

Create a matching Figma file with these frames:

1. Home / Hero
2. About Me
3. Skills
4. Case Study: Healthcare Member Journey
5. Case Study: Financial Advisor Dashboard
6. Case Study: Retail Product Discovery
7. Contact

Use the React UI as the visual reference for the Figma design.
